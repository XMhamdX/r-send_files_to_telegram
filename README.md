# سكريبت رفع الملفات إلى تيليجرام - دليل التثبيت

هذا الملف يحتوي على قائمة المكتبات المطلوبة والتعليمات اللازمة لتشغيل السكريبت بنجاح.

## المكتبات المطلوبة (Python Packages)
1. **Telethon**: المكتبة الأساسية للتفاعل مع تيليجرام.
2. **Cryptg**: مكتبة اختيارية تسرع عملية التشفير والرفع بشكل كبير جداً.

## المتطلبات الخارجية (External Tools)
*   **FFmpeg**: ضروري لمعالجة الفيديوهات وتحويلها. يجب تحميله وإضافته إلى مسار النظام (System PATH).

## أمر التثبيت (Installation Command)
لتثبيت جميع المكتبات المطلوبة بضغطة واحدة، قم بفتح شاشة الأوامر (Terminal) في مجلد السكريبت وتشغيل الأمر التالي:

```bash
pip install -r requirements.txt
```

أو يمكنك ببساطة تشغيل ملف `install_dependencies.bat` إذا كنت تستخدم ويندوز.
